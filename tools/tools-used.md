# Tools Used

## Linux Terminal

Purpose:
System administration and navigation.

Commands Used:
- pwd = Print working directory
- ls -al = Lists hidden files
- cd
- find <starting_point> -name <filename> = Finds the path to a specified file
- mkdir
- cat
- whoami = Prints the logged in user
- uname -a = Prints details about os, kernel version, and architecture
- df -h = Checks disk space human readable

## Windows Command Prompt

Purpose:
Basic networking and system administration tasks.

Commands Used:
- dir /a = Lists hidden files
- dir /s <filenamw> = Finds the path to a specified file
- type <filename> = Prints file contents
- hostname = Name of the computer
- systeminfo
- ipconfig
